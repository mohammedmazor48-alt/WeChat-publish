## ⚠️ 注意事项

1. **HTML文件中的图片引用**
   - 图片路径是相对路径: `cover_final.jpg`, `illustration_*.jpg`
   - 发布到公众号时需要:
     a) 上传图片到公众号素材库
     b) 替换HTML中的路径为公众号图片URL

2. **自动脚本依赖**
   - 需要 Python 3.10+
   - 需要 wechatpy, requests, python-dotenv
   - 需要正确的 AppID 和 AppSecret

3. **手动发布推荐**
   - 如果自动脚本运行失败
   - 建议手动复制HTML内容
   - 逐张上传图片
   - 这样格式控制更精确

## 🔧 故障排除

### 问题: 封面图文字显示乱码
**解决**: 系统缺少中文字体，可:
1. 安装字体文件到系统
2. 或修改 generate_images.py 使用系统已有字体

### 问题: 推送脚本报错
**解决**: 
1. 检查 .env 文件配置
2. 检查网络连接
3. 检查 AppID/AppSecret 是否正确
4. 尝试手动发布

### 问题: HTML在公众号显示异常
**解决**:
1. 公众号编辑器不支持某些CSS
2. 建议复制纯文本后重新排版
3. 图片单独上传插入
